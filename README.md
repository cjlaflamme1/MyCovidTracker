# About The Project

## Group Project 1

Create an interactive web app utilizing 2 API's, an unfamiliar CSS framework, and a polished UI that's free of alerts, confirms, and prompts. 


## Technologies Used
    * Basic: HTML, CSS, Javascript
    * JQuery
    * Materialize
    * API's:
        - Coronavirus COVID19 API
        - NY Times API
